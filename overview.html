Installation¶

RAMSTK can be installed in several ways. Firstly, RAMSTK is available from PyPi.

pip install ramstk

Or the source code can be checked-out from GitHub and installed the old-fashioned way.

git checkout https://github.com/ReliaQualAssociates/ramstk.git ramstk.git
cd ramstk.git
python setup.py install --user

where the –user switch is optional if you’re sure you want to install RAMSTK in a system-wide location.
