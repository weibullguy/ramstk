Contain the attributes and methods of an analysis manager.

This class manages the analyses for RAMSTK modules. Attributes of the analysis manager are:

Variables

  • _attributes (dict) – the dict used to hold the aggregate attributes for the hardware item being analyzed.

  • tree – the treelib Tree() used to hold a copy of the data manager’s tree. This do not remain in-sync automatically.

class ramstk.controllers.managers.RAMSTKMatrixManager(column_tables: Dict[str, List[Any]], row_table: object)[source]¶

Bases: object

The meta-class for all RAMSTK Matrix Managers.

The Matrix data model is an aggregate model of N x M cell data models. The attributes of a Matrix are:

Variables

  • _row_table (object) – the RAMSTK Program database table to use for the matrix rows. This is an SQLAlchemy object.

  • column_tables – a list of RAMSTK data table objects that comprise the columns. One table per matrix managed by an instance of the matrix manager.

  • dic_matrices – the dictionary containing all the matrices managed by an instance of the matrix manager.

  • n_row – the number of rows in the Matrix.

  • n_col – the number of columns in the Matrix.
