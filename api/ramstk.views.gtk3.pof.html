ramstk.views.gtk3.pof.workview module¶

The RAMSTK PoF Work View.

class ramstk.views.gtk3.pof.workview.PoF(configuration: ramstk.configuration.RAMSTKUserConfiguration, logger: ramstk.logger.RAMSTKLogManager)[source]¶

Bases: ramstk.views.gtk3.widgets.baseview.RAMSTKWorkView

Display PoF attribute data in the Work Book.

The WorkView displays all the attributes for the Physics of Failure Analysis (PoF). The attributes of a PoF Work View are:

Variables

  • _dic_column_masks (dict) – dict with the list of masking values for the PoF worksheet. Key is the PoF indenture level, value is a list of True/False values for each column in the worksheet.

  • _dic_headings (dict) – the dict with the variable headings for the first two columns. Key is the name of the PoF indenture level, value is a list of heading text.

  • _dic_keys (dict) –

  • _dic_column_keys (dict) –

  • _lst_labels (list) – the list of labels for the widgets on the work view. The PoF work stream module has no labels, but an empty list is required to prevent an AttributeError when creating the UI.

  • _module (str) – the name of the module.

  • _pixbuf (bool) – indicates whether or icons are displayed in the RAMSTKTreeView. If true, a GDKPixbuf column will be appended when creating the RAMSTKTreeView. Default is True.

  • _lst_callbacks (list) – the list of callback methods for the view’s toolbar buttons and pop-up menu. The methods are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_icons (list) – the list of icons for the view’s toolbar buttons and pop-up menu. The icons are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_mnu_labels (list) – the list of labels for the view’s pop-up menu. The labels are listed in the order they appear in the menu.

  • _lst_tooltips (list) – the list of tooltips for the view’s toolbar buttons and pop-up menu. The tooltips are listed in the order they appear on the toolbar or pop-up menu.
