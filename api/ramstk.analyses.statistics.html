ramstk.analyses.statistics.bounds module¶

Functions for performing calculations associated with statistical bounds.

ramstk.analyses.statistics.bounds.calculate_crow_bounds(n_failures, t_star, _lambda, beta, alpha, metric, data=2)[source]¶

Calculate confidence bounds based on Dr. Larry Crow’s methods.

Parameters

  • n_failures – the total number of failures in the data set.

  • t_star – termination time.

  • _lambda – the estimated scale parameter.

  • beta – the estimated shape parameter.

  • alpha – the desired confidence level of the bounds.

  • metric – indicates which metric the bounds are being calculated for. 1 = shape (beta) 2 = scale (lambda) 3 = cumulative failure intensity 4 = instantaneous MTBF (future)

  • data – the type of data set being passed. 1 = Type I or time terminated test 2 = Type II or failure terminated test (default)

Returns

_crow_l, _crow_u; the lower and upper bound on the metric.

Return type

tuple of floats

ramstk.analyses.statistics.bounds.calculate_fisher_bounds(metric, variance, alpha)[source]¶

Calculate the Fisher Information Matrix based confidence bounds.

Parameters

  • metric – the point estimate of the metric to calculate bounds for.

  • variance – the variance of the metric to calculate bounds for.

  • alpha – the confidence level of the calculated bounds.

Returns

_fisher_l, _fisher_u; the lower and upper Fisher bounds.

Return type

tuple of floats

ramstk.analyses.statistics.bounds.calculate_nhpp_mean_variance(n_failures, max_time, alpha, beta, metric=1)[source]¶

Calculate the variance for the NHPP and Crow-AMSAA models.

Parameters

  • n_failures – total number of failures in the data set.

  • max_time – total observation time or maximum failure time.

  • alpha – the point estimate of the alpha (scale) parameter.

  • beta – the point estimate of the beta (shape) parameter.

  • metric – the metric to calculate the variance for. - 1 = Cumulative MTBF. - 2 = Instantaneous MTBF.

Returns

the variance of the selected mean.

Return type

float

ramstk.analyses.statistics.bounds.calculate_variance_covariance(n_failures, max_time, alpha, beta)[source]¶

Calculate variance-covariance matrix for NHPP and Crow-AMSAA models.

Parameters

  • n_failures – total number of failures in the data set.

  • max_time – total observation time or maximum failure time.

  • alpha – the point estimate of the alpha (scale) parameter.

  • beta – the point estimate of the beta (shape) parameter.

Returns

the variance-covariance matrix for alpha and beta. It has the form:

[[Var(alpha), Cov(alpha, beta)],

[Cov(alpha, beta), Var(beta)]]

Return type

list of lists

ramstk.analyses.statistics.bounds.do_calculate_beta_bounds(minimum, likely, maximum, alpha)[source]¶

Calculate the mean, standard error, and bounds of the beta distribution.

These are the project management estimators, not exact calculations.

Parameters

  • minimum – the minimum expected value.

  • likely – most likely value.

  • maximum – the maximum expected value.

  • alpha – the desired confidence level.

Returns

_meanll, _mean, _meanul, _sd; the calculated mean, bounds, and standard error.

Return type

tuple of floats
