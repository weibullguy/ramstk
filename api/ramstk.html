class ramstk.configuration.RAMSTKUserConfiguration[source]¶

Bases: object

RAMSTK configuration class.

Attributes of the Configuration class are:

Variables

  • RAMSTK_FORMAT_FILE (dict) – Dictionary containing the path to the format files to use for various widgets. Keys are the name of the RAMSTK work stream module, values are the absolute path to the format file for that work stream module.

  • RAMSTK_COLORS (dict) – Dictionary containing the colors to use for various widgets.

  • RAMSTK_COM_INFO (dict) –

    Dictionary for the RAMSTK common database connection information. The information contained is:

    Key

    Information

    host

    Host name

    port

    Host port

    database

    Database name

    user

    User name

    password

    User password

  • RAMSTK_PROG_INFO (dict) –

    Dictionary for RAMSTK Program database connection information. The information contained is:

    Key

    Information

    host

    Host name

    port

    Host port

    database

    Database name

    user

    User name

    password

    User password

  • RAMSTK_TABPOS (dict) –

    Dictionary containing the location of tabs in the three main Gtk.Notebook() widgets. Can be one of:

    • Top

    • Bottom

    • Left

    • Right

    Key

    Notebook

    Default

    listbook

    Module Book

    top

    modulebook

    Work Book

    bottom

    workbook

    List Book

    bottom

  • RAMSTK_SEVERITY (dict) – Dictionary for failure severity categories.

  • RAMSTK_HAZARDS (dict) – Dictionary for potential hazards.

  • RAMSTK_REQUIREMENT_TYPES (dict) – Dictionary of requirement types.

  • RAMSTK_RPN_SEVERITY (dict) – Dictionary for RPN Severity categories.

  • RAMSTK_RPN_OCCURRENCE (dict) – Dictionary for RPN Occurrence categories.

  • RAMSTK_RPN_DETECTION (dict) – Dictionary for RPN Detection categories.

  • RAMSTK_MODULES (dict) –

    Dictionary of active modules in the open RAMSTK Program database. Where 1 = active and 0 = inactive. Keys are:

    • Function

    • Hardware

    • Requirements

    • Revision

    • Validation

  • RAMSTK_PAGE_NUMBER (list) – Dictionary indicating which page each RAMSTK module occupies in the ModuleBook.

  • RAMSTK_RISK_POINTS (list) –

    List for risk level cutoffs. Cutoffs are:

    Index

    Risk Level Cutoff Value

    0

    Low to medium

    1

    Medium to high

  • RAMSTK_HR_MULTIPLIER (float) – The failure rate multiplier. All failure rates will be multiplied by this value for display. This allows failure rates to display without using scientific notation. Set to one to use scientific notation. Default value is 1000000.0.

  • RAMSTK_MTIME (float) – The default mission time for new RAMSTK Programs.

  • RAMSTK_DEC_PLACES (int) – Number of decimal places to show in numerical results. Default value is 6.

  • RAMSTK_MODE_SOURCE (int) –

    Indicator variable used to determine which failure mode source to use. Sources are:

    1. FMD-97

    2. MIL-STD-338

  • RAMSTK_CONF_DIR (str) – Path to the directory containing configuration files used by RAMSTK.

  • RAMSTK_DATA_DIR (str) – Path to the directory containing data files used by RAMSTK.

  • RAMSTK_ICON_DIR (str) – Path to the directory containing icon files used by RAMSTK.

  • RAMSTK_LOG_DIR (str) – Path to the directory containing log files used by RAMSTK.

  • RAMSTK_PROG_DIR (str) – Path to the base directory containing RAMSTK Program database files. This is only used when the backend is SQLite3.

  • RAMSTK_GUI_LAYOUT (str) –

    Layout of the GUI to use. Possible options are:

    • basic - a single window embedded with the Module Book, Work Book,

      and List Book.

    • advanced - multiple windows; one each for the Module Book, Work

      Book, and List Book.

    Default value is basic.

  • RAMSTK_COM_BACKEND (str) –

    RAMSTK common database backend to use. Options are:

    • mysql (future)

    • sqlite

  • RAMSTK_BACKEND (str) –

    RAMSTK Program database backend to use. Options are:

    • mysql (future)

    • sqlite

  • RAMSTK_LOCALE (str) – The language locale to use with RAMSTK. Default value is en_US.

  • RAMSTK_OS (str) – The operating system RAMSTK is currently running on.

ramstk.ramstk module¶

The RAMSTK program manager.

class ramstk.ramstk.RAMSTKProgramManager[source]¶

Bases: object

The RAMSTK program manager class.

The RAMSTK program manager is responsible for managing all the analysis, data, and matrix managers associated with the RAMSTK program database that is currently open. The attributes of a RAMSTK program manager are:

Variables

  • dic_managers (dict) – a dict containing the instances of all the analysis, data, and matrix managers associated with the currently active run of RAMSTK. The first key is the workstream module name which has a dict as a value. The key of this second dict is the type of manager (analysis, data, matrix) and the value will be the instance of the applicable manager.

  • program_dao – the BaseDatabase() object that will connect to the RAMSTK program database.
