class ramstk.views.gtk3.requirement.listview.RequirementHardware(configuration: ramstk.configuration.RAMSTKUserConfiguration, logger: ramstk.logger.RAMSTKLogManager)[source]¶

Bases: ramstk.views.gtk3.widgets.baseview.RAMSTKListView

Display all the Requirement::Hardware matrix for the selected Revision.

The attributes of the Requirement::Hardware Matrix View are:

Variables

  • _module – the name of the module.

  • _lst_callbacks – the list of callback methods for the view’s toolbar buttons and pop-up menu. The methods are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_icons – the list of icons for the view’s toolbar buttons and pop-up menu. The icons are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_mnu_labels – the list of labels for the view’s pop-up menu. The labels are listed in the order they appear in the menu.

  • _lst_tooltips – the list of tooltips for the view’s toolbar buttons and pop-up menu. The tooltips are listed in the order they appear on the toolbar or pop-up menu.
