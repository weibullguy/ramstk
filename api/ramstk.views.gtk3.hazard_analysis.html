ramstk.views.gtk3.hazard_analysis.workview module¶

The RAMSTK GTK3 Function Work View.

class ramstk.views.gtk3.hazard_analysis.workview.HazOps(configuration: ramstk.configuration.RAMSTKUserConfiguration, logger: ramstk.logger.RAMSTKLogManager)[source]¶

Bases: ramstk.views.gtk3.widgets.baseview.RAMSTKWorkView

Display HazOps attribute data in the Work Book.

The WorkView displays all the attributes for the Hazards Analysis (HazOps). The attributes of a HazOps Work View are:

Variables

  • _module (str) – the name of the module.

  • _lst_callbacks (list) – the list of callback methods for the view’s toolbar buttons and pop-up menu. The methods are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_icons (list) – the list of icons for the view’s toolbar buttons and pop-up menu. The icons are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_mnu_labels (list) – the list of labels for the view’s pop-up menu. The labels are listed in the order they appear in the menu.

  • _lst_tooltips (list) – the list of tooltips for the view’s toolbar buttons and pop-up menu. The tooltips are listed in the order they appear on the toolbar or pop-up menu.

  • _hazard_id (int) – the ID of the currently selected hazard.
