ramstk.views.gtk3.fmea.workview module¶

The RAMSTK base FME(C)A Work View.

class ramstk.views.gtk3.fmea.workview.FMEA(configuration: ramstk.configuration.RAMSTKUserConfiguration, logger: ramstk.logger.RAMSTKLogManager)[source]¶

Bases: ramstk.views.gtk3.widgets.baseview.RAMSTKWorkView

Display FMEA attribute data in the Work Book.

The WorkView displays all the attributes for the Failure Mode and Effects Analysis (FMEA). The attributes of a FMEA Work View are:

Variables

  • _dic_column_masks (dict) – dict with the list of masking values for the FMEA worksheet. Key is the FMEA indenture level, value is a list of True/False values for each column in the worksheet.

  • _dic_headings (dict) – dict with the variable headings for the first two columns. Key is the name of the FMEA indenture level, value is a list of heading text.

  • _dic_keys (dict) –

  • _dic_column_keys (dict) –

  • _lst_control_type (list) – list containing the types of controls that can be implemented.

  • _lst_labels (list) – list containing the label text for each widget label.

  • _pixbuf (bool) – indicates whether or icons are displayed in the RAMSTKTreeView. If true, a GDKPixbuf column will be appended when creating the RAMSTKTreeView. Default is True.

  • _module (str) – the name of the module.

  • _lst_callbacks (list) – the list of callback methods for the view’s toolbar buttons and pop-up menu. The methods are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_icons (list) – the list of icons for the view’s toolbar buttons and pop-up menu. The icons are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_mnu_labels (list) – the list of labels for the view’s pop-up menu. The labels are listed in the order they appear in the menu.

  • _lst_tooltips (list) – the list of tooltips for the view’s toolbar buttons and pop-up menu. The tooltips are listed in the order they appear on the toolbar or pop-up menu.

  • _dic_missions (dict) – dict containing all this missions associated with the selected Revision.

  • _dic_mission_phases (dict) – dict containing all the mission phases associated with each mission in _dic_missions.

  • _item_hazard_rate (float) – hazard rate of the Hardware item associated with the FMEA.
