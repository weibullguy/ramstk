Capacitor Work View Panels.

class ramstk.views.gtk3.hardware.components.capacitor.AssessmentInputPanel[source]¶

Bases: ramstk.views.gtk3.hardware.components.panels.RAMSTKAssessmentInputPanel

Display Capacitor assessment input attribute data.

The Capacitor assessment input view displays all the assessment inputs for the selected capacitor. This includes, currently, inputs for MIL-HDBK-217FN2 parts count and part stress analyses. The attributes of a Capacitor assessment input view are:

Variables

  • _dic_quality (dict) – dictionary of MIL-HDBK-217 capacitor quality levels. Key is capacitor subcategory ID; values are lists of quality levels.

  • _dic_specifications (dict) – dictionary of capacitor MIL-SPECs. Key is capacitor subcategory ID; values are lists of specifications.

  • _dic_styles (dict) – dictionary of capacitor styles defined in the MIL-SPECs. Key is capacitor subcategory ID; values are lists of styles.

  • _lst_labels (list) – list of label text to display for the capacitor MIL-HDBK-217 input parameters.

  • _lst_widgets – the list of widgets to display in the panel. These are listed in the order they should appear on the panel.

  • cmbConfiguration – select and display the configuration of the capacitor.

  • cmbConstruction – select and display the method of construction of the capacitor.

  • cmbSpecification – select and display the governing specification of the capacitor.

  • cmbStyle – select and display the style of the capacitor.

  • txtCapacitance – enter and display the capacitance rating of the capacitor.

  • txtESR – enter and display the equivalent series resistance.

class ramstk.views.gtk3.hardware.components.semiconductor.AssessmentResultPanel[source]¶

Bases: ramstk.views.gtk3.hardware.components.panels.RAMSTKAssessmentResultPanel

Display semiconductor assessment results attribute data.

The semiconductor assessment result view displays all the assessment results for the selected semiconductor. This includes, currently, results for MIL-HDBK-217FN2 parts count and part stress methods. The attributes of a semiconductor assessment result view are:

Variables

  • txtPiT – displays the temperature factor for the semiconductor.

  • txtPiA – displays the application factor for the semiconductor.

  • txtPiC – displays the construction factor for the semiconductor.

  • txtPiI – displays the forward current factor for the semiconductor.

  • txtPiM – displays the matching network factor for the semiconductor.

  • txtPiP – displays the power degradation factor for the semiconductor.

  • txtPiR – displays the power rating factor for the semiconductor.

  • txtPiS – displays the electrical stress factor for the semiconductor.

class ramstk.views.gtk3.hardware.components.switch.AssessmentInputPanel[source]¶

Display Switch assessment input attribute data in the RAMSTK Work Book.

The Switch assessment input view displays all the assessment inputs for the selected switch. This includes, currently, inputs for MIL-HDBK-217FN2. The attributes of a switch assessment input view are:

Variables

  • _dic_applications (dict) – dictionary of switch applications. Key is switch subcategory ID; values are lists of applications.

  • _dic_construction (dict) – dictionary of switch construction methods. Key is switch subcategory ID; values are lists of construction methods.

  • _dic_contact_forms (dict) – dictionary of switch contact forms. Key is switch subcategory ID; values are lists of contact forms.

  • cmbApplication – select and display the switch application.

  • cmbConstruction – select and display the switch construction method.

  • cmbContactForm – select and display the switch contact form.

  • txtNCycles – enter and display the number of switch cycles/hour.

  • txtNElements – enter and display the number of switch wafers.
