class ramstk.views.gtk3.hardware.workview.AssessmentInputs(configuration: ramstk.configuration.RAMSTKUserConfiguration, logger: ramstk.logger.RAMSTKLogManager)[source]¶

Bases: ramstk.views.gtk3.widgets.baseview.RAMSTKWorkView

Display Hardware assessment input attribute data.

The Hardware assessment input view displays all the assessment inputs for the selected Hardware item. This includes, currently, inputs for MIL-HDBK-217FN2 and NSWC-11. The attributes of a Hardware assessment input view are:

Variables

  • _lst_labels (list) – the text to use for the assessment input widget labels.

  • _module (str) – the name of the module.

  • _dic_assessment_input (dict) – dictionary of component-specific AssessmentInputs classes.

  • _hardware_id (int) – the ID of the Hardware item currently being displayed.

  • _hazard_rate_method_id (int) – the ID of the hazard rate method used for Hardware item.

  • _lst_callbacks (list) – the list of callback methods for the view’s toolbar buttons and pop-up menu. The methods are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_icons (list) – the list of icons for the view’s toolbar buttons and pop-up menu. The icons are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_mnu_labels (list) – the list of labels for the view’s pop-up menu. The labels are listed in the order they appear in the menu.

  • _lst_tooltips (list) – the list of tooltips for the view’s toolbar buttons and pop-up menu. The tooltips are listed in the order they appear on the toolbar or pop-up menu.
