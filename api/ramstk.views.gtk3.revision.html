ramstk.views.gtk3.revision.moduleview module¶

RAMSTK Revision GTK3 module view.

class ramstk.views.gtk3.revision.moduleview.ModuleView(configuration: ramstk.configuration.RAMSTKUserConfiguration, logger: ramstk.logger.RAMSTKLogManager)[source]¶

Bases: ramstk.views.gtk3.widgets.baseview.RAMSTKModuleView

Display Revision attribute data in the RAMSTK Module Book.

The Revision Module View displays all the Revisions associated with the connected RAMSTK Program in a flat list. The attributes of a Revision Module View are:

Variables

  • _module – the name of the module.

  • _lst_mnu_labels – the list of labels for the view’s pop-up menu. The labels are listed in the order they appear in the menu.

  • _lst_tooltips – the list of tooltips for the view’s toolbar buttons and pop-up menu. The tooltips are listed in the order they appear on the toolbar or pop-up menu.

ramstk.views.gtk3.revision.workview module¶

The RAMSTK GTK3 Revision Work View.

class ramstk.views.gtk3.revision.workview.GeneralData(configuration: ramstk.configuration.RAMSTKUserConfiguration, logger: ramstk.logger.RAMSTKLogManager)[source]¶

Bases: ramstk.views.gtk3.widgets.baseview.RAMSTKWorkView

Display general Revision attribute data in the RAMSTK Work Book.

The Revision Work View displays all the general data attributes for the selected Revision. The attributes of a Revision General Data Work View are:

Variables

  • _module (str) – the name of the module.

  • _tablabel (str) – the text to display on the tab’s label.

  • _tabtooltip (str) – the text to display as the tab’s tooltip.

  • _lst_callbacks (list) – the list of callback methods for the view’s toolbar buttons and pop-up menu. The methods are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_icons (list) – the list of icons for the view’s toolbar buttons and pop-up menu. The icons are listed in the order they appear on the toolbar and pop-up menu.

  • _lst_mnu_labels (list) – the list of labels for the view’s pop-up menu. The labels are listed in the order they appear in the menu.

  • _lst_tooltips (list) – the list of tooltips for the view’s toolbar buttons and pop-up menu. The tooltips are listed in the order they appear on the toolbar or pop-up menu.
